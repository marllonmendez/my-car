## Sobre
Projeto criado com base em uma aplicação simples para trabalho da faculdade.

## Como usar

<h4>1. Clonagem</h4>

```bash
git clone https://github.com/marllonmendez/MyCar.git
```

<h4>2. Compilação</h4>

```bash
# Windows
javac -d bin src\main\java\car\*.java src\main\java\*.java

# Linux
javac -d bin src/main/java/car/*.java src/main/java/*.java
```

<h4>3. Execução</h4>

```bash
java -cp bin Main
```
